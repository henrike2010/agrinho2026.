# 🌿 EcoPlantio - Portal de Agricultura Sustentável

O **EcoPlantio** é um projeto de site institucional focado em conscientização ambiental e disseminação de práticas de cultivo agrícola sustentáveis. O objetivo é apresentar soluções de manejo que equilibrem a produção de alimentos com a preservação ambiental.

## 🚀 Tecnologias Empregadas
- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização moderna, responsiva e layout baseado em Grid.

## 💡 Práticas Abordadas no Projeto
- Rotação de Culturas
- Sistemas Agroflorestais
- Adubação Orgânica e Compostagem
